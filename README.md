# Sudoku App

A Sudoku game built with Flutter to practice Flutter development and improve problem-solving skills.

## Features

- 9×9 interactive Sudoku grid
- Number input using number pad
- Move validation with feedback
- Pencil mode for notes
- Undo and erase functionality
- Hint system for selected cells
- Row, column, and 3×3 box highlighting
- State management using Provider
